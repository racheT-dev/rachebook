# rachebook Блокнот с распознаванием рукописного текста (iOS)

Приложение заметок с интеграцией Yandex Vision API для автоматической оцифровки рукописных текстов.

## Настройка

1. Зарегистрируйтесь в Yandex Cloud, создайте сервисный аккаунт и получите API-ключ и folderId.
2. Откройте `Models/YandexVisionService.swift` и вставьте ключ и folderId в соответствующие поля.
3. Соберите и запустите проект в Xcode 14+ на симуляторе или устройстве с iOS 13+.

## Функции
- Создание, редактирование и удаление текстовых заметок
- Поиск по содержимому и тегам
- Сортировка по дате создания / изменения
- Сканирование рукописного текста с камеры или из галереи
- Автоматическое создание заметки с распознанным текстом
